CÁC CÂU LỆNH HAY DÙNG TRONG GIT
- git init: khởi tạo git trong một repository
-git remote -v: xem url của remote repo(từ xa)
- git remote add origin <url>: để có thể thêm, commit và push đến remote repo của mình
- git remote set-url origin <url>:dùng khi đổi remote repo
- git clone<url>: lấy project về
- git branch: liệt kê tất cả các branch(nhánh) trong local của bạn
- git branch <tên branch>: tạo nhánh mới
- git checkout <name>: sẽ chuyển sang một nhánh đã tồn tại khác.
- git checkout -b <name>: tạo nhánh mới và ngay lập tức chuyển sang nhánh đó.
- git merge <tên branch>: hợp nhất branch đó vào branch chính-master của bạn
- git pull origin <branch>: kéo những thay đổi mới nhất về
- git satus: biết những file nào đã bị thay đổi và file nào được theo dõi.
- git dif -stat: biết mỗi file đã thay đổi bao nhiêu dòng code.
